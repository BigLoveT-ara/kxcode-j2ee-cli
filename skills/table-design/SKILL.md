# 生成文档规范
## 生成产物
不需要你基于任何一个数据库语言来生成SQL语句，只是需要你生成出一个MD的表格内容分别包括
- 表名、表备注、表说明
- 枚举说明：如果表格中出现枚举类型的值，请在表头描述中对枚举值进行说明
- 关联说明：如果表有关联关系需要说明关联字段和关联情况
- 字段表：需要包括字段名、字段备注、字段类型（通用类型如String 、Int等）、非空、唯一约束
  
## 生成规范

- 表名：使用小写字母+下划线分割的组合，表名已tb_开头
- 字段名：使用小写字母+下划线分割的组合，避开type、status等可能保留字，如果要表示该含义时添加前缀，如`task_status`,`task_type`.
- 字段备注：必须包含字段备注和说明
- 外键：不需要定义外键，如果有关联关系的需要在字段备注和表头中标出关联表以及`1:1 1:n n:n`的关系
- 主键：主键非指定的特殊情况，一般采用自增类型，字段名为`id`。
- 在非指定的特殊情况，都需要为表创建`is_deleted`的逻辑删除字段、`create_time`、`create_by`、`update_time`、`update_by`的创建/修改时间、和创建修改用户。
