# 数据对象设计规范
 
## DAO 对象规范
- `*DAO.java` 与数据库表结构**一一对应**，用作结构化数据库映射
- 若 DAO 中包含关联查询字段或非持久化字段，必须使用相应注解标注（如 MyBatis-Plus 的 `@TableField(exist = false)`）
 
## 对象复用原则（禁止过度创建）
- API 入参对象的属性若是某 `*DAO.java` 的**子集**，直接复用该 DAO，不得额外新建对象；但需在接口文档中标注各字段是否必填
- API 返回值对象的属性若是某 `*DAO.java` 的**子集**，直接复用该 DAO，不得额外新建对象；但需在接口文档中标注各字段是否必填
 
## 新建对象的条件
仅当无法用现有对象满足需求时，才允许新建，并按用途命名：
- `*VO.java`：视图展示对象（接口返回）
- `*DTO.java`：数据传输对象（接口入参 / 跨层传递）
- `*POJO.java`：通用简单对象（无明确归属时）
